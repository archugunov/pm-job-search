# Rubric: Open-ended UX critique

After applying the structured rubrics (TONE, lint, spec criteria), step back and read the transcript as a real user would.

Questions to consider:
- Did any prompt confuse the simulated user? Look for "what do you mean", "I'm not sure what you're asking", or terse "skip" responses to questions that should have been answered.
- Did the assistant feel preachy, lecture-y, or moralistic at any point?
- Did the assistant hedge unnecessarily ("I think", "maybe", "perhaps")?
- Were there dead ends — places where the next action was unclear?
- Did the closing message sound generic ("Great! Let me know if you need anything else.") or specific to the context?
- Did the assistant repeat itself across turns?
- Did the persona's character break (e.g. an "experienced senior PM" being walked through basic concepts)?

## How to report findings under this rubric

Free-form observations. 3-7 bullets max. Each bullet: one observation + transcript line reference. Do not invent issues to fill the section — if the transcript is clean here, write "No open critiques" and move on.
